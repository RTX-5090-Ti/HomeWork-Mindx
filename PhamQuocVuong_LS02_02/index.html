Các thuộc tính CSS được dùng ở hình trên

  1. CSS reset : để reset CSS mặc định của trình duyệt
  2. width: để chỉnh chiều rộng của các thẻ
  3. height: để chỉnh chiều cao các thẻ
  4. color: chỉnh màu cho chữ (#fcf7f5, #545454)
  5. background-color: chỉnh mày nền (#db5c40)
  6. background-image: dùng để chèn ảnh
  7. background-position: tuỳ chỉnh vị trí ảnh (left, right, top, bottom)
  8. text-align: tuỳ chỉnh vị chí của đoạn text
  9. text-decoration: tuỳ chỉnh chữ (in đậm, loại bỏ chỉ mục của thẻ danhh sách)
  10. line-height: tuỳ chỉnh khoản cách các dòng
  11. font-size: chỉnh cỡ chữ
  12. font-weight: chỉnh độ đậm nhạt cho chữ
  13. display: flex chỉnh giao diện cho wed
  14. posotion: tuỳ chỉnh vị trí cho các thẻ, các khối
  15. left, right, top, bottoom: tuỳ chỉnh vị trí khi set position
  16. CSS Pseudo class: các tuỳ chỉnh khi đưa con trỏ tới (hover, active)
  17. outline: none -> xoá đường viền xanh xung quanh thẻ input
  18. shadow: tạo bóng
